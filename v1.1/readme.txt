v1.1 - readme.txt

Η έκδοση v1.1 αποθηκεύει την διάταξη του κάθε επιπέδου σε αρχείο κειμένου.

1. Αναγκαίες βιβλιοθήκες:

Για την εκτέλεση του κώδικα είναι αναγκαίες οι παρακάτω βιβλιοθήκες:
- Matplotlib
- time
- random
- datetime

Η εγκατάσταση τους μπορεί να γίνει μέσω του pip ή εναλλακτικά μέσω του package manager του λειτουργικού σας συστήματος αν η διαχείρηση των libraries δεν γίνεται από το pip.

2. Εκτέλεση του κώδικα:

Η εκτέλεση του κώδικα γίνεται ανοίγωντας τον φάκελο upatras-autonavigation/v1.1 και εκτελώντας το αρχείο main.py με χρήση της εντολής `python3 main.py`.
