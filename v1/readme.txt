1. Αναγκαίες βιβλιοθήκες:

Για την εκτέλεση του κώδικα είναι αναγκαίες οι παρακάτω βιβλιοθήκες:
- Matplotlib
- time
- random
- copy

Η εγκατάσταση τους μπορεί να γίνει μέσω του pip (π.χ. pip install flet) ή εναλλακτικά μέσω του package manager του λειτουργικού σας συστήματος αν η διαχείρηση των libraries δεν γίνεται από το pip. Ορισμένες βιβλιοθήκες είναι ήδη προεγκατεστημένες στις περισσότερες εγκαταστάσεις της Python.

2. Εκτέλεση του κώδικα:

Η εκτέλεση του κώδικα γίνεται ανοίγοντας τον φάκελο upatras-autonavigation/v1 και εκτελώντας το αρχείο main.py με χρήση της εντολής `python3 main.py`.
